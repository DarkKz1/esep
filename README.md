# Esep — отчёт донору из полевых заметок за минуту

**Живой деплой:** https://esep-ngo.netlify.app
**Хакатон:** nFactorial «Startup Platform для партнёра», 17–20 июля 2026. Соло-проект.

## Партнёр и сфера

Выбранная сфера — **партнёрская экосистема Chevron в Казахстане** ([communitypartnership.lovable.app](https://communitypartnership.lovable.app)): 16 социальных программ, 14 партнёрских НКО, программа **Capacity Building**, в чьём брифе на 2026 год прямо записана задача «измерение импакта совместных проектов».

## Проблема

НКО, живущая на гранты, обязана регулярно отчитываться перед донором. Реальность полевой работы — хаос: цифры охвата в WhatsApp-чатах координаторов, анкеты в Google Forms, наблюдения в заметках. Сведение этого в структурированный отчёт занимает у менеджера программы дни — и это время, отнятое у самой программы. Показатель «32 000 охваченных людей» на дашборде Chevron кто-то собирал руками именно из такого хаоса.

## Решение

**Esep** (қаз. «отчёт») — **портал импакта с двумя сторонами**:

**Сторона НКО** (`#/submit`): вставляешь сырые заметки как есть → AI структурирует их в отчёт (резюме, деятельность, таблица охвата, результаты, голоса, проблемы) → отправляешь донору одним нажатием.

**Сторона донора** (главная): живой агрегированный дашборд по всем поступившим отчётам — суммарный охват, программы, партнёры, города — как статичная витрина на сайте Chevron, только наполняется в реальном времени.

Три принципиальные вещи, которые отличают Esep от «обёртки над ChatGPT»:

1. **Ни одной выдуманной цифры.** Модель жёстко ограничена фактами из заметок (форсинг структуры через Anthropic tool-use); всё отсутствующее уходит в `dataGaps`, а не в метрики.
2. **Линза качества данных и рисков** — уникальный слой для донора. Портал считает «качество данных» (доля отчётов без пробелов и высоких рисков) и выносит наверх блок **«Требует внимания донора»**: где НКО недосдали данные, где флаги риска (напр. несогласованный перерасход бюджета). Это то, что донор раньше вылавливал вручную.
3. **Раздел «Чего не хватает»** — отчёт сам говорит полевой команде, какие данные досчитать. Инструмент превращается из «писалки текста» в дисциплину сбора данных.

Отчёт на русском или английском (международным донорам нужен EN).

## Стек

- **Frontend:** Vite + React 18, хеш-роутер без зависимостей, ручной CSS (paper-and-ink дизайн-система), marked для markdown
- **Backend:** Netlify Functions —
  - `extract.mjs`: Anthropic API (Claude Sonnet) с форсингом строгого JSON через tool-use → структурированный отчёт
  - `reports.mjs`: хранилище на Netlify Blobs, автосев демо-данными при первом обращении
- **Данные:** 14 реалистичных отчётов реальных программ-партнёров Chevron (суммарный охват ≈ 32 000) — портал живой сразу, без ручного наполнения
- **Фидбэк:** Netlify Forms (сабмиты проверяемы в дашборде — пруфы для хакатона)
- **Деплой:** Netlify

## Как смотреть демо

1. Открой https://esep-ngo.netlify.app — портал донора со всеми отчётами и риск-линзой.
2. «Сдать отчёт (НКО)» → «Заполнить демо-данными» → «Разобрать заметки» → «Отправить донору».
3. Новый отчёт появляется на портале с подсветкой, счётчик охвата растёт.

> Примечание: живая AI-экстракция требует наличия кредитов на Anthropic-аккаунте. Портал донора и вся агрегация работают всегда (данные в Blobs).

## Запуск локально

```bash
npm install
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env
netlify dev        # http://localhost:8888
```

Без `netlify dev` (чистый Vite) фронт поднимется, но генерация требует функции — используйте `netlify dev` или задеплоенную версию.

## Структура

```
netlify/functions/generate.mjs   # SSE-passthrough к Anthropic, промпт отчёта, фоллбэк моделей
src/App.jsx                      # весь UI: ввод → стриминг → экспорт → форма фидбэка
src/demoData.js                  # демо-заметки вымышленной НКО (кнопка «Заполнить демо-данными»)
src/styles.css                   # editorial paper-and-ink дизайн-система
```

## Фидбэк пользователей

Собирается через встроенную форму «Оставить отзыв» (Netlify Forms) и прямые интервью с сотрудниками НКО. Сводный документ с пруфами — в материалах сдачи хакатона.
