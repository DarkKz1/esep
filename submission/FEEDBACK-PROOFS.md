# Esep — пруфы фидбэка пользователей (шаблон)

> Заполняется по мере поступления фидбэка. Требование ТЗ: минимум 5 разных реальных
> пользователей, релевантных сфере (НКО / грантовая отчётность / CSR), фидбэк ПОСЛЕ
> реального использования продукта, в проверяемом виде (скриншоты, записи, формы).

**Каналы сбора (проверяемые):**
1. Встроенная форма «Отзыв» на esep-ngo.netlify.app → Netlify Forms
   (дашборд: app.netlify.com/projects/esep-ngo/forms — экспорт CSV, у каждого сабмита серверный timestamp).
2. Скриншоты переписки (Telegram/WhatsApp) — прикладывать с датой.
